매크로 프로그램 방지

매크로프로그램 사용을 방지하기 위하여 관리자가 지정한 횟수이상 저장을 시도하면 화면에 나타나는 문자열을 입력하고 계속 사용 하여야 함.

문자열 오류 횟수가 5회 이상 이면 자동 로그아웃되고 문자열 입력이 올바를 경우 이전 요청이 자동 수행됨

새로고침(F5) 사용 불가

보안상의 이유로 브라우저의 새로고침(F5) 기능 사용시 로그아웃됨 (화면 내 새로고침 Refresh 버튼 사용)

팝업차단 해제

팝업차단 설정이 해제되지 않았을 경우 팝업차단을 해제

인터넷 익스플로러(IE) : 우측 상단 메뉴 오픈 -> 설정 -> 인터넷옵션 -> 개인정보 -> 팝업차단 체크 해지 또는 사이트 메뉴에 추가

크롬(Chrome) : 우측 상단 메뉴 오픈 -> 설정 -> 설정검색창에 "팝업" 검색 ->사이트설정 -> 팝업및 디렉션 -> 팝업허용 또는 사이트 추가

파이어폭스(Firefox) : 우측상단 메뉴 오픈 -> 설정 > 개인정보 및 보안 > 팝업 창 차단 체크 해지 또는 예외목록에 등록

팝업 설정 및 해제는 브라우저, 버전별로 방법이 다를 수 있으니 해당 브라우저 공식사이트 또는 검색을 통해 확인 바람

모니터 해상도

수강신청시스템은 1920*1080 해상도에 최적화되어 있으며 그 이상의 해상도로 화면 설정

해상도가 작아 정상적으로 보이지 않을 경우 브라우저 확대,축소 기능 으로 사용자 환경에 맞게 조절 후 사용

수강신청전 미리 확인하여 수강신청시 지장 없도록 유의하기 바람

브라우저

크롬, 파이어폭스 브라우저에 최적화 되있음

해상도가 작아 정상적으로 보이지 않을 경우 브라우저 확대,축소 기능으로 적절히 조절 후 사용

인터넷 익스플로러 10이하 버전을 사용하시는 분들은 정상적으로 동작하지 않을 수 있으며, 보안상의 이유로도 10버전 이상으로 업그레이드하시기를 권장

이화여대 어플 다운로드 및 개인정보 업데이트

대기수강 교과목 PUSH 알림을 위하여 “이화여자대학교 HOME” 어플 다운로드 후, PUSH 알림 설정. ‘유레카>개인정보변경’에서 개인정보(특히 휴대전화 번호) 업데이트 필수

Macro Prevention

If students try to click excessively more than the number of times that system manager set up, putting key word page would come up on the screen for macro prevention. When typing letters incorrectly for 5 times, users will be logged out. When typing letters correctly, course registration process will proceed.

Prevention for Multi Login

If students login with same student ID on more than two devices simultaneously (including PC and mobile), only last login is validate to use and rest of them would be logged out forcibly.

Refresh(F5) Prevention

For security purposes, when students try to click “Refresh(F5)” button of your browser, they would be logged out automatically. Please only use 새로고침 Refresh button on your page.

Please enable the POP UP service on your PC.

팝업차단 설정이 해제되지 않았을 경우 팝업차단을 해제

Internet Explorer(IE): Set-up ⟶ Toos ⟶ Internet Option ⟶ Personal Information ⟶ Enable the POP UP service or add the Sugang Site

Chrome : Set-up ⟶ Toos ⟶ seach “pop up” in the search box ⟶ set up the site ⟶POP UP and Direcrtion ⟶ Enable the POP UP service or add the Sugang Site

Firefox : Set-up ⟶ Toos ⟶ Personal Informaion and Security ⟶ Enable the POP UP service or add the Sugang Site

팝업 설정 및 해제는 브라우저, 버전별로 방법이 다를 수 있으니 해당 브라우저 공식사이트 또는 검색을 통해 확인 바람

Browser

Sugang portal system is optimized on Chrome or firefox browsers.

If you have a low resolution, you should adjust your monitor.

If you use Internet Explorer(IE) under 10.0 version, your sugang system will not be worked, and strongly recommended to upgrade to over 10.0 version for security.

EWHA HOME APP download and update personal information

In order to get the alarm for waiting courses, you should install EWHA HOME APP through Apple or Play store, and enable the PUSH alarm service. Also update your personal information(especially your mobile phone number) on “Eureka > Personal Information.”
